# C-Study

임베디드 시스템 개발을 위한 C 언어 기초 및 심화 학습 기록입니다.

## 학습 목표
- 포인터 및 메모리 구조 이해
- 구조체 기반 데이터 설계
- 함수 포인터 활용
- 스택과 힙 메모리 동작 원리 이해

## 디렉토리 구조

- 01_pointer : 포인터 기본 및 배열과의 관계
- 02_struct : 구조체 사용 및 설계
- 03_memory : 스택/힙 메모리 구조
- 04_function_pointer : 함수 포인터 및 콜백 개념

## 학습 방식
- 개념 학습 후 직접 코드 작성
- 동작 원리 정리
- 예제 확장 실습
